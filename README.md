# English Learning Portal

Interactive English learning platform with pronunciation, vocabulary, and grammar guides.

## 🌐 Live Site

https://ntviet2409.github.io/learning-portal/

## ✨ Features

- **IPA Pronunciation Guide** - Learn correct pronunciation with International Phonetic Alphabet
- **Common Mistakes** - Avoid typical English learner errors
- **Vocabulary Builder** - Expand your word knowledge systematically
- **Grammar Lessons** - Master English grammar rules
- **ChatGPT Learning Tips** - AI-assisted learning strategies

## 🔧 Development

```bash
# Local development server
python3 -m http.server 8000
# Visit: http://localhost:8000

# Deploy (automatic via GitHub Pages)
git add .
git commit -m "Update content"
git push origin main
```

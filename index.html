Welcome! Here's how to get started:

This app is designed for Vietnamese learners. It has everything you need: pronunciation guide (IPA), 1000 essential words, and grammar with Vietnamese explanations. Follow this learning path for the best results:

  1. Start with IPA Pronunciation — Learn the sound symbols first. Click the speaker icons to hear each sound. This helps you read pronunciation for every word.
  2. Set your daily target — Start with 10 words/day above. Learn 5 new words + review 5 old ones. Consistency beats speed!
  3. Learn vocabulary by topic — Start with "Greetings", then "Numbers". Click "Today's Words" for your daily dose. Mark words as "Learned" when you're confident.
  4. Test yourself — Use Flashcards to review, then Quiz to test. Aim for 80%+ before moving to the next topic.
  5. Study grammar alongside — After learning 50-100 words, start reading the grammar section. The Vietnamese translations will help you understand the English tenses.
